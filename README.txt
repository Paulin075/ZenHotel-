Pour que ce site fonctionne correctement, vous devez ajouter la base de données ZenHotel à votre serveur PHP local. Cela permettra d'assurer la bonne gestion et l'affichage des données nécessaires pour l'application.
 https://paulin075.github.io/ZenHotel-/
https://github.com/Paulin075/ZenHotel-.git
Suivez bien les instructions pour que toute les parties marches chez vous vu que la base de donnée est en local vous devrez la téléchargé et l'exporté dans mysql que vous soyez sur xamp ou wamp(De préférence) ensuite mettre le tout dans un fichier que vous mettrez dans le dossier www si vous êtes sur wamp et http si vous êtes sur xamp
